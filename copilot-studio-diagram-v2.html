1 Build — Authoring Environment
🧩

Copilot Studio Designer

A low-code / no-code web canvas where makers build conversational copilots visually — no deep AI expertise required. Configure topics, test conversations, and publish — all in one place.

Visual canvas No-code / low-code Built-in testing One-click publish
2 Core Building Blocks
💬

Topics

Conversation flows triggered by user phrases or intents. Each topic contains nodes — questions, messages, conditions, and actions — that guide the dialogue.

Trigger phrases Branching logic Variables
⚡

Actions

Extend capabilities by calling Power Automate flows, custom connectors, or Azure services to perform real-world tasks inside a conversation.

Power Automate HTTP requests Connectors
📚

Knowledge Sources

Ground generative AI responses in authoritative content — documents, websites, SharePoint, and Dataverse — to reduce hallucinations and improve accuracy.

SharePoint Public URLs Files / PDFs
3 AI Engine — Intelligence Layer
🤖

Generative AI (Azure OpenAI)

Powers natural language understanding, fallback responses, and generative answers when no exact topic matches the user's message.

GPT-powered Grounded responses
🎯

NLU & Intent Recognition

Analyses user messages, matches them to the closest topic using built-in language models, and manages conversation context across multiple turns.

Intent matching Context tracking
🔐

Authentication & Security

Supports AAD / SSO, role-based access, data loss prevention policies, and tenant-level governance through Power Platform admin centre.

AAD / SSO DLP policies RBAC

🔗

Native integration with Power Automate (flows), Power Apps (embedding), and Dataverse for structured data storage and retrieval.

Power Automate Power Apps Dataverse

Teams SharePoint Azure services

External Systems

REST APIs SAP / Salesforce ServiceNow
5 Publish — Deployment Channels
💼

One-click publish as a Teams app. Appears in the Teams app store for internal employees with SSO pre-wired — no extra sign-in required.

SSO pre-wired Internal app store
🌍

Website / Web Chat

Embed a chat widget on any web page with a single script tag. Customise colours, greetings, and avatars to match your brand.

Script embed Fully branded

Messenger / Slack IVR / Telephony Direct Line API
